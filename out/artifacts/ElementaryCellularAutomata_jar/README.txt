In this folder is a JAR file containing the code to run the Elementary Cellular Automata simulator. User needs to have java installed in order to run the code. From terminal or command line, user can enter the following line to start the program:

java -jar ElementaryCellularAutomata.jar

Alternatively, Mac users can enter ./RUN.sh in terminal to run the program (if this does not work, try typing `chmod +x RUN.sh` and trying again). Windows users can double click on RUN.bat to run the program.
